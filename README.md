# ys-fw-micro

[YS](https://github.com/yskit/ys-mutify) 架构之基于微服务模型的应用架构。

# Install

```shell
npm i --save ys-fw-micro
```

# Usage

请在项目中的`package.json`指定`framework`属性即可。

```javascript
{
  "name": "xxxx",
  "version": "1.0.0",
  "framework": "ys-fw-micro"
}
```
